# ⚡ Mini Arduino Starter Kit

Welcome to the **[Mini Arduino Starter Kit](https://thinkingrobot.in/products/mini-arduino-starter-kit-thinking-robot-compact-beginner-pack)** manual!

This is the compact beginner pack from Thinking Robot to get you started with embedded programming quickly and affordably.

[![Mini Arduino Starter Kit](https://res.cloudinary.com/drwys1ksu/image/upload/v1771040578/61-6PmrhyML_rdpfmj.jpg)](https://thinkingrobot.in/products/mini-arduino-starter-kit-thinking-robot-compact-beginner-pack)

## Key Concepts to Learn

- **Digital Read/Write**: Control LEDs and read button presses.
- **Analog Read/Write**: Read potentiometers and dim LEDs with PWM.
- **Sensors & Actuators**: Add an MPU sensor to detect motion!
